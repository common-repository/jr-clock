== Description ==

This plugin allows you to display one of 10 flash clocks on your blog as a widget!

You can simply select which clock you want from a list of them into the Settings --> JR Clock menu and you'll be ready to go!


It's very easy to setup the plugin - Simply enable it, alter the settings and enable the widget!

== Installation ==

1. Upload `jr-clock.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Clock Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
